# Micro Change Watch

Daily public-page monitoring for founders, operators, consultants, and small teams.

Send a few public URLs you care about — competitor pricing pages, changelogs, docs, release notes, status pages, marketplace listings, or GitHub releases. Micro Change Watch checks them daily and sends a concise human-readable digest when something meaningful changes.

## Launch offer

- **Starter — $9/month**: monitor up to 3 public URLs, daily digest.
- **Standard — $19/month**: monitor up to 10 public URLs, daily digest.
- **48-hour Watch — $10 one-off**: monitor up to 5 public URLs around a launch, incident, competitor move, or release window.

## What you get

Each report includes:

- changed URL;
- short summary;
- evidence snippet / before-after diff;
- fetch errors, if any;
- no-change list so you know the watch ran.

Example report: [`docs/sample-report.md`](docs/sample-report.md)

## Good fits

- Competitor pricing and plan pages.
- SaaS changelogs and product-update pages.
- API/docs pages you depend on.
- GitHub release pages and repo documentation.
- Public status pages.
- Marketplace or directory listings where normal public fetching is allowed.

## Not a fit

No private dashboards, logins, cookies, credentials, personal data monitoring, captcha-heavy sites, scraping-sensitive targets, bypassing access controls, or trading/financial advice workflows.
