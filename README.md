# Micro Change Watch

Stop manually checking competitor pricing, vendor changelogs, API docs, release notes, and status pages.

Send 3–10 public URLs. Micro Change Watch checks them daily and sends a concise digest when something meaningful changes — with the link, before/after evidence, and an operator-reviewed note on why it may matter.

## Launch offer

- **48-hour Watch — $10 one-off**: monitor up to 5 public URLs around a launch, incident, competitor move, or release window.
- **Standard — $19/month**: monitor up to 10 public URLs, daily digest.
- **Starter — $9/month**: monitor up to 3 public URLs, daily digest.

## What you get

Each report includes:

- changed URL;
- short operator-reviewed summary when a change is meaningful;
- evidence snippet / before-after diff;
- fetch errors, if any;
- no-change list so you know the watch ran.

Example report: [`docs/sample-report.md`](docs/sample-report.md)

## Good fits

- “Tell me when Competitor X changes pricing or plan limits.”
- “Tell me when Vendor Y updates their API changelog/docs.”
- “Tell me when this GitHub project cuts a new release.”
- “Tell me when a public status page or incident page changes.”
- “Tell me when my marketplace/directory listing changes.”

## Not a fit

No private dashboards, logins, cookies, credentials, personal data monitoring, captcha-heavy sites, scraping-sensitive targets, bypassing access controls, or trading/financial advice workflows.

Work starts after payment or explicit approval for a trial/demo.
